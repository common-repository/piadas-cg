== Description ==

Plugin de post automático de piadas do portal Click Grátis.

== Installation ==

1. Upload the folder `piadas-cg` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure it in the Settings admin menu

== Frequently Asked Questions ==

= Como desabilito o plugin? =

Coloque o valor 0 no intervalo de horas.
